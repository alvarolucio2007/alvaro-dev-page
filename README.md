# 🚀 Meu Portfólio Pessoal

Bem-vindo ao repositório do meu portfólio pessoal! Este é o lugar onde compartilho meus principais projetos, habilidades, experiências e formas de entrar em contato comigo. 

O site foi construído focado em alta performance, SEO e uma experiência de usuário fluida.

🔗 **Link do projeto:** [Visite meu portfólio](https://alvarolucio2007.github.io/alvaro-dev-page/)

---

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando o ecossistema moderno do desenvolvimento web:

*   **[SvelteKit](https://kit.svelte.dev/):** Framework principal para roteamento, performance e animações fluidas.
*   **[Tailwind CSS](https://tailwindcss.com/):** Para uma estilização rápida, responsiva e utilitária.
*   **TypeScript:** Garantindo tipagem estática e segurança no código.
*   **GitHub Pages:** Hospedagem gratuita e automatizada diretamente pelo repositório.
